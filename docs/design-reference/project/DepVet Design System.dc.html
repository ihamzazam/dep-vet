03 · Risk-signal language

The loudest thing on screen. Encoded three ways — color + shape + label — so it survives color-blindness and grayscale. Never use color alone.

Critical
◆ diamond · #FF4530
CVEs (high/crit), typosquats, malicious install-scripts. Blocks the verdict.
Warning
▲ triangle · #FFB020
Moderate CVEs, abandonment, outdated majors. Review before shipping.
Healthy
● circle + ✓ · #B8FF5C
Current, maintained, no advisories. The quiet majority.
